## 环境设置

1. 创建虚拟环境:
```bash
python -m venv venv
source venv/bin/activate  # Windows系统: venv\Scripts\activate
```

2. 安装依赖:
```bash
pip install -r requirements.txt
```

3. 运行数据库迁移:
```bash
python manage.py migrate
```

4. 创建超级用户:
```bash
python manage.py createsuperuser
```

5. 启动开发服务器:
```bash
python manage.py runserver
```

访问 http://127.0.0.1:8000/ 查看网站。
